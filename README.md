# QuickTerm - 极简 Tauri 终端

一个基于 Tauri v2 + React + TypeScript 构建的跨平台极简终端模拟器。旨在提供极致简洁、响应迅速的命令行体验。

## ✨ 已实现功能

### 1. 核心终端功能
- **跨平台命令执行**：通过 Rust 后端调用系统 Shell 执行命令，支持 macOS、Linux 和 Windows。
- **目录切换 (cd)**：
  - 完美支持 `cd ~`、`cd -` 以及相对/绝对路径切换。
  - 自动展开波浪号 `~` 到用户主目录。
  - 记录工作目录历史，支持快速返回。
- **命令别名 (Alias)**：内置常用别名（如 `ll`, `la`, `gs`, `gp` 等），支持极简指令（如 `..`, `...`）。

### 2. UI/UX 美化
- **Oh My Zsh 风格**：借鉴经典终端美化方案，提供清爽的界面视觉。
- **极简路径显示**：
  - 仅显示当前所在的目录名，避免长路径占用过多空间。
  - 主目录智能显示为 `~`。
- **Git 状态集成**：在提示符中实时显示当前所在的 Git 分支（如 `(main)`）。
- **深色模式**：内置深色主题，针对代码高亮和输出结果进行了优化。

### 3. 便捷操作
- **自动聚焦**：启动即聚焦输入框，随时待命。
- **清屏支持**：支持 `clear` 命令及 `Ctrl + L` 快捷键。
- **智能滚动**：输出内容自动滚动到底部。

---

## 🛠 技术架构

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Tauri v2 (Rust)
- **Communication**: Tauri IPC (invoke)
